AXIVAI is a resilience infrastructure platform that empowers school districts, CFOs, policymakers, and utilities to make confident, data-driven decisions about electrifying America's school bus fleet. Built for public-sector trust and auditability.

## Core Modules

| Module | Description |
|--------|-------------|
| **Research Engine** | Automated, citation-aware intelligence with weekly updates |
| **TCO Calculator** | Planning-grade total cost of ownership with scenario analysis |
| **Solution Explorer** | Interactive problem-to-solution mapping with evidence trails |
| **Trust Center** | Compliance, accessibility, and AI governance transparency |
| **AXIVAI Chat** | Grounded Q&A with full citation and scope awareness |

## Local Development

```bash
# Clone the repository
git clone https://github.com/quartermasters/axivai-platform.git
cd axivai-platform

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Start development server
npm run dev
```

## Security

**Never commit `.env` files.** All secrets must be configured via environment variables. See [.env.example](.env.example) for required configuration.
